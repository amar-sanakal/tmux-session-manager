tmux Session Manager

I had written a small rudimentary Python script to create tmux sessions based on a config file where I could provide a name for the session along with a list of servers that I needed to open for that named session. It lacked the ability to save and/or restore such a session but it served it's purpose. However, when some of my colleagues wanted to use it they brought up some of the missing features and I thought of extending it into it's own project.
